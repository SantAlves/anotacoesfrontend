Tipos de listas em HTML

Listas não ordenas

  • Listas são importantes para muitos fins nos sites;
  • Podemos até criar um menu a partir de uma lista;
  • As não ordenadas são criadas pela tag ul;
  • Cada item na lista é representado pela tag li;
  • Podemos ter listas dentro de listas(aninhadas)

Vamos ver na prática!

  • item 1
  • item 2
  • item 3
    • item 3.1
    • item 3.2
    • item 3.3

Listas ordenas

  1. Listas ordenadas são interessantes para quando há um procedimento ou passos a serem seguidos;
  2. Exemplo: receita de algum alimento;
  3. Agora utilizamos a tag OI;
  4. E os itens continuam sendo a tag li;

Vamos ver na prática!

  1. item 1
  2. item 2
  3. item 3
    1. item 3.1
    2. item 3.2
    3. item 3.3
